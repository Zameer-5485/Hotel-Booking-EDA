# Hotel Booking Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on hotel booking data to identify booking patterns, cancellation trends, customer behavior, and seasonal demand insights.

The analysis focuses on understanding factors affecting hotel bookings and cancellations using Python-based data analysis techniques.

---

## Objectives
- Analyze hotel booking patterns
- Identify cancellation trends
- Compare City Hotel and Resort Hotel performance
- Understand seasonal booking behavior
- Explore customer demographics and booking preferences
- Generate business insights from the dataset

---

## Dataset Information
The dataset contains hotel booking records with features such as:
- Hotel type
- Booking dates
- Cancellation status
- Average Daily Rate (ADR)
- Country
- Lead time
- Customer details

---

## Key Insights
- City Hotels have higher cancellation rates compared to Resort Hotels.
- Booking demand increases during peak holiday months.
- ADR changes significantly across seasons.
- Certain countries contribute more to booking cancellations.
- Online travel agents contribute heavily to bookings.

---

## Business Recommendations
- Implement stricter cancellation policies during peak seasons.
- Offer promotional discounts during low-demand periods.
- Improve pricing strategies based on seasonal demand.
- Focus marketing efforts on high-performing customer segments.
